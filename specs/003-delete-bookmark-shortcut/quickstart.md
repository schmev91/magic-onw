# Quickstart: Deleting Bookmarks

This guide covers how to use the new keyboard shortcut to manage your bookmarks.

## Steps

1. **Open the list view**:
   Run the `onw` command without arguments.
   ```bash
   onw
   ```

2. **Select a bookmark**:
   Use the `Up` and `Down` arrow keys to highlight the alias you wish to remove.

3. **Press Delete**:
   Hit the `Delete` key on your keyboard.

4. **Confirm the deletion**:
   A prompt will appear: `Are you sure you want to delete alias [name]? [y/n]`.
   - Press `y` or `Enter` to confirm.
   - Press `n` or `Esc` to cancel.

5. **Success**:
   The bookmark will disappear from the list, and a confirmation message will be shown.
